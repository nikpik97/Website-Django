I'm hosting this website on a raspberry pi using the Nginx/Gunicorn stack as the http server and Python's Django framework for the backend. This webpage is going to serve (pun intended) as my first real foray into web development. My goal is to add an "about me" page, a blog, a portfolio, and a contact me form as well as a well-designed backend to handle all of that. Check back in a few weeks and, hopefully, you'll be seeing a more beautiful, featureful, responsive website!
